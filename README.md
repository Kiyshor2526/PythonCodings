# Python Programs Collection 🐍

This repository contains beginner-friendly Python programs created for learning and practice. It includes pattern printing programs, number-based programs, and a simple ATM simulation project.

## 📂 Programs Included

### ⭐ Pattern Programs
- Diamond Pattern
- Reverse Star Pattern
- Reverse Same Number Pattern
- Reverse Descending Number Pattern
- Right-Aligned Star Triangle
- Right-Aligned Reverse Star Triangle

### 🔢 Number Programs
- Find Largest Digit from the Number
- Length of the Number
- Sum of Digits of a Number
- Multiplication Table

### 🏧 Mini Project
- ATM Simulator

## 🛠️ Technologies Used
- Python 3

## ▶️ How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Kiyshor2526/<repository-name>.git
   ```

2. Open the project folder:
   ```bash
   cd <repository-name>
   ```

3. Run any Python program:
   ```bash
   python filename.py
   ```
